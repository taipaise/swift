### UI구현 : 스토리 보드

설명
- 슬라이드 바를 통해서 1 ~ 60초 사이의 시간을 설정할 수 있다.
- start 버튼을 누르면 초가 1초씩 감소한다.
- 0초가 되면 화면 색이 변하고 system sound를 재생한다.
- reset 버튼을 눌러 초기화할 수 있다.
