### UI 구현 : 스토리 보드 / 코드

설명
- 화면 이동에 대해 공부
- present를 통해 코드로만 구현된 뷰 컨트롤러로 이동
- present를 통해 스토리 보드로 구현된 뷰 컨트롤러로 이동
- 스토리 보드로 구현된 뷰 컨트롤러에서 다음 뷰 컨트롤러로 세그 연결
- 스토리 보드로 구현된 뷰 컨트롤러의 버튼에서 다음 뷰 컨트롤러로 세그 연결
